Deploy on GitHub Pages

  1. Create a new GitHub repo (public is fine for demo).
  2. Upload all files in this ZIP to the repository root.
  3. In GitHub: Settings → Pages → Source: Deploy from a branch, Branch: main, Folder: / (root).
  4. Wait for the link (e.g., https://yourname.github.io/repo/). Open it and log in using password demo123.

Everything is static — no build steps required.
